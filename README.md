# dong5854's blog

Next.js 16 (App Router) 기반 개인 기술 블로그

## 기술 스택

- **Framework**: Next.js 16 (App Router, Turbopack)
- **Content**: MDX + Velite
- **Styling**: Tailwind CSS
- **Search**: kbar
- **Deploy**: Vercel
- **Analytics**: Cloudflare Web Analytics

## 시작하기

```bash
yarn install
yarn dev
```

## 스크립트

| 명령어 | 설명 |
|---|---|
| `yarn dev` | 개발 서버 (Velite watch 포함) |
| `yarn build` | 프로덕션 빌드 |
| `yarn serve` | 프로덕션 서버 |
| `yarn lint` | ESLint 실행 |
